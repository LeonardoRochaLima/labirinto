Descrição dos Elementos de Jogo:
 - Mapa: Imagem do campo onde o avatar irá atuar para resolver o puzzle.
 - Avatar: Ator que será responsável por se movimentar pelo mapa.
 - Para movimentar o avatar do jogo, use as setas.

Regras:
 - Quanto maior a quantidade de movimentos, menor será sua pontuação.
 - Não é possível atravessar as paredes do labirinto.

Condições de Vitória:
 - Encontre o caminho para a saída do labirinto.
